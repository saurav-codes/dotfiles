# dotfiles

## Installation

```bash
# install required packages
sudo apt-get install build-essential -y
sudo apt-get install libncurses5-dev libgtk2.0-dev libatk1.0-dev libcairo2-dev libx11-dev libxpm-dev libxt-dev -y
sudo apt-get install liblua5.3-dev python3-dev ruby-dev -y
# above command may give error, if so, try this:
sudo apt-get install -f
# then run the above command again
